# Project: Lottery Number Generator

## Overview

A simple web application that generates 6 unique random numbers for a lottery game.

## Features

*   **Lottery Number Generation:** Generates 6 unique random numbers between 1 and 45.
*   **User Interface:**
    *   A button to trigger the number generation.
    *   A section to display the generated numbers.
*   **Styling:**
    *   Clean and modern design.
    *   Responsive layout.
