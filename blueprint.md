# Lotto Number Generator

## Overview

A simple and fun tool to generate random numbers for various lottery games. Get your lucky numbers with a single click!

## Features

-   **Random Number Generation:** Generates 6 unique random numbers between 1 and 45.
-   **Animations:** Numbers are displayed with a smooth fade-in animation.
-   **Responsive Design:** The layout adapts to different screen sizes.
-   **Internationalization:** The website is available in English and Korean, with a language switcher to select the preferred language.

## Pages

-   **Home:** The main page with the number generator.
-   **About:** Information about the project.
-   **Contact:** A contact form for inquiries.
-   **Privacy Policy:** The privacy policy of the website.

## Tech Stack

-   HTML5
-   CSS3
-   JavaScript (ES6+)

## Design

-   **Color Scheme:** The design uses a clean and modern color palette with shades of blue, yellow, red, and green for the lotto balls.
-   **Typography:** The site uses a clean, sans-serif font for readability.
-   **Layout:** The layout is simple and centered, with a clear focus on the number generator.

## Internationalization Plan

-   **Korean Content:** All English HTML files have been duplicated and translated into Korean. The Korean files are named with a `-ko` suffix (e.g., `index-ko.html`).
-   **Language Switcher:** A dropdown menu has been added to the navigation bar of all pages to allow users to switch between English and Korean.
-   **Language Detection:** The website automatically detects the user's browser language and redirects them to the appropriate version of the site. The user's language preference is stored in `localStorage`.
